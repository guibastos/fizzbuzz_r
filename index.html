FizzBuzz

This is the game of FizzBuzz...For each number if it’s not divisible by 3 or 5, you simply say the number. If it’s divisible by 3, instead of the number you say “Fizz”. If it’s divisible by 5, you say “Buzz”. And finally if it’s divisible by both 3 and 5, you say “FizzBuzz”. So, counting from one to fifteen, you’d say: “One, two, Fizz, four, Buzz, Fizz, seven, eight, Fizz, Buzz, eleven, Fizz, thirteen, fourteen, FizzBuzz”. Enter a number (integer, positive) and let's count!
